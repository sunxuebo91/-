## 📋 核心功能

### 1. HTTP 请求工具 (`utils/request.js`)

**功能：**
- ✅ 封装 `wx.request`
- ✅ 自动添加 JWT Token
- ✅ 处理 401 错误（Token 过期）
- ✅ 统一错误处理

**API：**
```javascript
const { publicRequest, authenticatedRequest, request } = require('./utils/request.js');

// 公开请求（无需 Token）
publicRequest({ url: '/auth/login', method: 'POST', data: {...} });

// 认证请求（需要 Token）
authenticatedRequest({ url: '/auth/me', method: 'GET' });

// 自动判断（有 Token 则认证，无 Token 则公开）
request({ url: '/resumes', method: 'GET' });
```

---

### 2. 认证服务 (`services/auth.js`)

**功能：**
- ✅ 账号密码登录
- ✅ Token 验证
- ✅ 用户信息管理
- ✅ 登出

**API：**
```javascript
const authService = require('./services/auth.js');

// 登录
await authService.login(username, password);

// 验证 Token
const isValid = await authService.validateToken();

// 获取用户信息
const userInfo = authService.getLocalUserInfo();

// 检查是否已登录
const isLoggedIn = authService.isLoggedIn();

// 登出
authService.logout();
```

---

### 3. 登录页面 (`pages/login/index.js`)

**功能：**
- ✅ 账号密码登录（调用"安得家政"API）
- ✅ 微信手机号登录（原有功能，保留）
- ✅ Token 自动验证
- ✅ 登录状态检查

**核心逻辑：**
```javascript
// 页面加载时检查登录状态
async onLoad() {
  await this.checkLoginStatus();  // 验证 Token
  this.loadMe();                   // 加载用户信息（微信登录）
}

// 账号密码登录
async onAccountLogin() {
  const response = await authService.login(username, password);
  authService.saveAuthData(response.data);
  wx.switchTab({ url: '/pages/home/index' });
}
```

## 🔄 登录流程

### 账号密码登录流程

```
┌─────────────────────────────────────────────────────────────┐
│  1. 用户输入账号密码                                          │
│     - 账号：zhangsan                                         │
│     - 密码：123456                                           │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  2. 调用 authService.login(username, password)              │
│     - 封装请求参数                                           │
│     - 调用 publicRequest()                                  │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  3. 发送 HTTP 请求                                           │
│     POST https://crm.andejiazheng.com/api/auth/login        │
│     { username: "zhangsan", password: "123456" }            │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  4. CRM 后台验证                                             │
│     - 验证用户名和密码                                        │
│     - 生成 JWT Token                                         │
│     - 返回用户信息                                           │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  5. 保存认证数据                                             │
│     - access_token: JWT Token                               │
│     - userInfo: { id, name, role, permissions, ... }        │
└─────────────────────────────────────────────────────────────┘
                          ↓
┌─────────────────────────────────────────────────────────────┐
│  6. 跳转到首页                                               │
│     wx.switchTab({ url: '/pages/home/index' })              │
└─────────────────────────────────────────────────────────────┘
```

## 🔐 数据存储

### 本地存储结构

| Key | 类型 | 说明 | 示例 |
|-----|------|------|------|
| `access_token` | String | JWT Token（主键） | `eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...` |
| `token` | String | JWT Token（备用键） | 同上 |
| `userInfo` | Object | 用户信息（主键） | `{ id, name, role, permissions, ... }` |
| `user_info` | Object | 用户信息（备用键） | 同上 |
| `openid` | String | 微信 OpenID | `oXXXXXXXXXXXXXXXXXXXXXXXXXXX` |

### 用户信息结构

```javascript
{
  id: "user_123",                    // 用户 ID
```
