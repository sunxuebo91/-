# 账号密码登录功能测试说明

## 功能概述

在原有的手机号快捷登录基础上，新增了账号密码登录功能，包括：
1. 账号密码登录
2. 账号注册
3. 两种登录方式可以共存

## 部署步骤

### 1. 部署云函数

在微信开发者工具中：
1. 右键点击 `cloudfunctions/userService` 目录
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

### 2. 初始化数据库

云函数会自动创建 `accounts` 集合，用于存储账号密码信息。

数据库结构：
```javascript
// accounts 集合
{
  _id: "xxx",
  username: "testuser",      // 账号（4-20位字母或数字）
  password: "123456",         // 密码（实际项目中应加密）
  nickname: "测试用户",       // 昵称
  openid: "xxx",              // 微信 openid
  createdAt: Date,            // 创建时间
  lastLoginAt: Date           // 最后登录时间
}
```

## 测试流程

### 测试注册功能

1. 打开小程序，进入登录页面
2. 滚动到底部，找到"账号密码登录"区域
3. 点击"立即注册"按钮
4. 进入注册页面，填写信息：
   - 账号：4-20位字母或数字，例如 `testuser`
   - 密码：至少6位，例如 `123456`
   - 确认密码：与密码一致
   - 昵称：任意昵称，例如 `测试用户`
5. 点击"立即注册"按钮
6. 注册成功后会提示，并自动返回登录页面

### 测试登录功能

1. 在登录页面的"账号密码登录"区域
2. 输入刚才注册的账号和密码
3. 点击"账号密码登录"按钮
4. 登录成功后会跳转到首页

### 测试表单验证

**注册页面验证：**
- 账号为空 → 提示"请输入账号"
- 账号格式错误 → 提示"账号必须是4-20位字母或数字"
- 密码少于6位 → 提示"密码至少6位"
- 两次密码不一致 → 提示"两次密码不一致"
- 昵称为空 → 提示"请输入昵称"
- 账号已存在 → 提示"账号已存在"

**登录页面验证：**
- 账号为空 → 提示"请输入账号"
- 密码为空 → 提示"请输入密码"
- 账号不存在 → 提示"账号不存在"
- 密码错误 → 提示"密码错误"

## 页面展示

### 登录页面
- 顶部：头像选择 + 昵称输入（用于手机号登录）
- 中部：手机号快捷登录按钮
- 底部：账号密码登录表单
  - 账号输入框
  - 密码输入框
  - 账号密码登录按钮
  - "还没有账号？立即注册"链接

### 注册页面
- 标题："注册账号"
- 表单：
  - 账号输入框
  - 密码输入框
  - 确认密码输入框
  - 昵称输入框
- 立即注册按钮
- "已有账号？立即登录"链接

## 数据库查询

可以在云开发控制台查看数据：
1. 打开云开发控制台
2. 进入数据库
3. 查看 `accounts` 集合，可以看到注册的账号信息
4. 查看 `users` 集合，可以看到登录后的用户信息

## 注意事项

1. **密码安全**：当前密码是明文存储，实际项目中应该使用 bcrypt 等加密算法
2. **账号唯一性**：系统会检查账号是否已存在，避免重复注册
3. **多设备登录**：同一个账号可以在不同设备登录，会更新 openid
4. **用户信息同步**：登录后会将账号信息同步到 users 集合
