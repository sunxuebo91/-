# 安得褓贝 · 高级感海报设计方案

> 版本：v2.0  日期：2026-04-01  
> 目标：从"PPT风格"升级为"杂志封面"级别的视觉质感

---

## 一、设计理念

**核心方向：「杂志封面」风格**

高端品牌（奢侈品、高端人力资源、五星酒店员工卡）的共同特点：
- 照片铺满全屏，无变形
- 渐变遮罩自然融合，无硬切割线
- 信息层叠浮在图上，层次分明
- 字体大小差异明显，主次清晰

## 三、分层结构（从下到上叠加）

```
┌──────────────────────────────────────┐
│  Layer 1: 全出血人像照片（背景层）     │ 0 ~ 640px

│  · 取景锚点：水平居中 + 垂直偏上15%  │
├──────────────────────────────────────┤
│  Layer 2: 渐变遮罩（叠在照片上）      │ 约 y:350 ~ 640px
│  · createLinearGradient 四锚点       │

│  · 45%  → rgba(15, 5, 30, 0.55)     │
│  · 75%  → rgba(15, 5, 30, 0.88)     │
│  · 100% → rgba(15, 5, 30, 0.97)     │

│  Layer 3: 信息文字区                  │ 约 y:430 ~ 590px

│       "扫码了解详情" 11px            │
└──────────────────────────────────────┘
```

## 六、关键技术实现要点

### 6.1 Cover 模式（解决变形问题）
```js
// 计算 cover 缩放比（取较大值确保填满）
const scaleX = W / imgW;
const scaleY = H / imgH;
const scale = Math.max(scaleX, scaleY);
const drawW = imgW * scale;
const drawH = imgH * scale;
// 水平居中，垂直偏上15%（让脸出现在画面上半）
const dx = (W - drawW) / 2;
const dy = Math.min(0, (H - drawH) * 0.15);
ctx.drawImage(photoImg, dx, dy, drawW, drawH);
```

### 6.2 渐变遮罩
```js
const grad = ctx.createLinearGradient(0, H * 0.55, 0, H);
grad.addColorStop(0,    'rgba(15,5,30,0)');
grad.addColorStop(0.45, 'rgba(15,5,30,0.55)');
grad.addColorStop(0.75, 'rgba(15,5,30,0.88)');
grad.addColorStop(1,    'rgba(15,5,30,0.97)');
ctx.fillStyle = grad;
ctx.fillRect(0, 0, W, H);
```

### 6.3 胶囊标签
```js
function drawPill(ctx, text, x, y, fontSize = 13) {
  ctx.font = `${fontSize}px sans-serif`;
  const tw = ctx.measureText(text).width;
  const ph = 24, pr = 12; // 高度、水平内边距
  const pw = tw + pr * 2;
  // 背景
  ctx.fillStyle = 'rgba(255,255,255,0.12)';
  roundRect(ctx, x, y - ph / 2, pw, ph, ph / 2);
  ctx.fill();
  // 边框
  ctx.strokeStyle = 'rgba(255,255,255,0.4)';
  ctx.lineWidth = 0.8;
  roundRect(ctx, x, y - ph / 2, pw, ph, ph / 2);
  ctx.stroke();
  // 文字
  ctx.fillStyle = '#ffffff';
  ctx.textBaseline = 'middle';
  ctx.fillText(text, x + pr, y);
  return pw + 10; // 返回占用宽度（含间距），用于连续排列
}
```

## 七、与旧版对比

| 对比项 | 旧版（廉价感） | 新版（高级感） |
|--------|--------------|--------------|
| 照片处理 | 强制拉伸，人脸变形 | Cover裁剪，比例自然 |
| 信息区背景 | 纯色紫色块，硬切割 | 渐变遮罩，自然融合 |
| 姓名字号 | 13px，与其他信息同大 | 28px，视觉主角 |
| 职业/年限显示 | 纯文字 `\|` 分隔 | 胶囊标签，精致感 |
| 月薪 | 混在文字串里 | 金色独立行，一眼看到 |
| 二维码 | 白色方块直接贴 | 圆角白卡，名片质感 |
| 整体色调 | 亮紫+白，像外卖App | 深紫渐变，奢侈品感 |
| 海报高度 | 560px | 640px，信息区呼吸感更好 |
