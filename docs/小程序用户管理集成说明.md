# 小程序用户管理集成说明

> **完成时间**: 2026-01-21  
> **功能**: 小程序用户信息同步到 CRM 后端

---

## ✅ 已完成功能

### 1. **自动登录（app.js）**

小程序启动时自动调用 CRM 后端登录接口，使用 OpenID 静默登录。

**实现位置**: `miniprogram/app.js`

**调用流程**:
```
小程序启动 (onLaunch)
  ↓
调用 wx.login 获取 code
  ↓
调用云函数 userService.getOrCreateMe 获取 OpenID
  ↓
调用 CRM 后端登录接口 POST /api/miniprogram-users/login
  ↓
保存用户信息到 globalData 和本地存储
```

**接口调用**:
```javascript
wx.request({
  url: 'https://crm.andejiazheng.com/api/miniprogram-users/login',
  method: 'POST',
  data: { openid: 'oXXXX-xxx' }
});
```

**响应数据**:
```json
{
  "success": true,
  "data": {
    "userId": 1,
    "phone": "13800138000",
    "nickname": "微信用户",
    "avatar": "https://...",
    "hasPhone": true,
    "message": "登录成功"
  }
}
```

**关键字段**:
- `hasPhone`: 是否已授权手机号（`true`/`false`）
- 如果 `hasPhone` 为 `false`，可以在首页引导用户授权手机号

---

### 2. **手机号授权注册（login/index.js）**

用户授权手机号后，自动调用 CRM 后端注册接口，同步用户信息。

**实现位置**: `miniprogram/pages/login/index.js`

**调用流程**:
```
用户点击"手机号快捷登录"
  ↓
上传头像到云存储（如果有）
  ↓
调用云函数 userService.loginByPhone 解密手机号
  ↓
调用 CRM 后端注册接口 POST /api/miniprogram-users/register
  ↓
保存用户信息到 globalData 和本地存储
  ↓
登录成功，返回上一页
```

**接口调用**:
```javascript
wx.request({
  url: 'https://crm.andejiazheng.com/api/miniprogram-users/register',
  method: 'POST',
  data: {
    openid: 'oXXXX-xxx',
    phone: '13800138000',
    nickname: '微信用户',
    avatar: 'cloud://xxx.jpg',
    gender: 0,  // 0未知 1男 2女
    city: '北京市',
    province: '北京'
  }
});
```

**响应数据**:
```json
{
  "success": true,
  "data": {
    "userId": 1,
    "isNewUser": true,
    "message": "注册成功"
  }
}
```

## 📱 使用方式

### 获取用户信息

在任何页面中，可以通过以下方式获取用户信息：

```javascript
// 方式1：从全局变量获取
const app = getApp();
const userInfo = app.globalData.userInfo;

// 方式2：从本地存储获取
const userInfo = wx.getStorageSync('crmUserInfo');

console.log('用户ID:', userInfo.userId);
console.log('手机号:', userInfo.phone);
console.log('昵称:', userInfo.nickname);
console.log('是否已授权手机号:', userInfo.hasPhone);
```

### 判断是否需要引导授权

```javascript
const app = getApp();
const userInfo = app.globalData.userInfo;

if (!userInfo || !userInfo.hasPhone) {
  // 引导用户授权手机号
  wx.showModal({
    title: '提示',
    content: '为了更好地为您服务，请授权手机号',
    success: (res) => {
      if (res.confirm) {
        wx.navigateTo({ url: '/pages/login/index' });
      }
    }
  });
}
```
