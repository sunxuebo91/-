# 育儿宝典模块使用说明

## 📋 功能概述

在首页（resumeList页面）的Banner轮播下方添加了"育儿宝典"文章模块，以卡片形式展示育儿知识文章。

## 🎯 实现内容

### 1. API服务封装 (`miniprogram/services/article.js`)

封装了两个核心接口：

#### 获取文章列表
```javascript
articleService.getArticleList({
  page: 1,
  pageSize: 10,
  status: 'published',  // 只获取已发布的文章
  keyword: '关键词'      // 可选，搜索关键词
})
```

**接口地址**: `GET https://crm.andejiazheng.com/api/articles?status=published&page=1&pageSize=10`

**返回数据格式**:
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "_id": "文章ID",
        "title": "文章标题",
        "author": "作者",
        "source": "来源",
        "summary": "摘要",
        "coverImage": "封面图URL",
        "imageUrls": ["图片1", "图片2"],
        "contentHtml": "<p>富文本内容</p>",
        "publishedAt": "2024-01-01T00:00:00.000Z",
        "viewCount": 100
      }
    ],
    "total": 50,
    "page": 1,
    "pageSize": 10,
    "totalPages": 5
  }
}
```

#### 获取文章详情
```javascript
articleService.getArticleDetail(articleId)
```

**接口地址**: `GET https://crm.andejiazheng.com/api/articles/:id`

**特点**:
- ✅ 无需认证（公开接口）
- ✅ 自动处理错误
- ✅ 统一日志输出

### 2. 页面集成 (`miniprogram/pages/resumeList/`)

#### 数据结构
```javascript
data: {
```

- `loadArticles()`: 加载文章列表（默认加载6篇）

## 🚀 使用方法

### 页面加载时自动获取文章
```javascript
onLoad() {
  this.reload();        // 加载简历列表
  this.loadArticles();  // 加载文章列表
}
```

### 点击文章卡片
```javascript
goArticleDetail(e) {
  const id = e.currentTarget.dataset.id;
  // 跳转到文章详情页
  wx.navigateTo({
    url: `/pages/articleDetail/index?id=${id}`
  });
}
```

## 🎨 样式定制

可以通过修改 `index.wxss` 中的以下类来调整样式：

- `.articles-section` - 整体容器
- `.articles-grid` - 网格布局
- `.article-card` - 单个卡片
- `.article-cover` - 封面图
- `.article-title` - 标题
- `.article-meta` - 元信息（作者、阅读量）
