# 小红书风格文章详情页改造说明

## 📱 改造效果

完全按照**小红书 APP** 的文章详情页风格实现：

### 核心特点

| 特性 | 说明 |
|------|------|
| **顶部大图** | 全屏大图展示（750rpx 高度） |
| **多图轮播** | 支持左右滑动切换图片 |
| **图片指示器** | 右下角显示 "1/5" |
| **圆角卡片** | 内容区域向上覆盖，圆角设计 |
| **作者信息** | 头像 + 姓名 + 时间 + 阅读量（横向排列） |
| **沉浸式布局** | 图片全屏，内容卡片浮在图片上 |

---

## 🎨 核心功能

### 1. **多图轮播**
- 使用 `<swiper>` 组件实现图片轮播
- 支持左右滑动切换图片
- 支持循环播放
- 图片高度固定为 500rpx（约 250px）

### 2. **图片来源优先级**
```
1. imageUrls 数组（API返回的多图字段）
   ↓
2. coverImage 字段（单张封面图）
   ↓
3. 从 HTML 内容中提取所有图片
```

### 3. **图片指示器**
- 右下角显示 "1/5" 样式的指示器
- 半透明黑色背景 + 白色文字
- 毛玻璃效果（backdrop-filter）

### 4. **图片预览**
- 点击图片可放大查看
- 支持左右滑动切换
- 支持双指缩放

### 5. **作者信息卡片**
- **头像**：渐变色圆形头像，显示作者名字首字母
- **姓名**：作者名称
- **时间**：发布时间
- **阅读量**：眼睛图标 + 阅读数

## 🚀 使用方法

### 数据格式要求

**后端 API 返回的文章数据应包含以下字段**：

```javascript
{
  "_id": "文章ID",
  "title": "文章标题",
  "author": "作者名称",
  "summary": "文章摘要",
  "content": "文章正文（HTML格式）",
  "publishedAt": "2026-01-27T10:00:00.000Z",
  
  // 多图字段（推荐）
  "imageUrls": [
    "https://example.com/image1.jpg",
    "https://example.com/image2.jpg",
    "https://example.com/image3.jpg"
  ],
  
  // 或者单图字段（兼容）
  "coverImage": "https://example.com/cover.jpg"
}
```

### 图片数量限制

- **最多显示 9 张图片**（小红书风格）
- 如果 `imageUrls` 数组超过 9 张，会自动截取前 9 张
- 如果只有 1 张图片，也会正常显示（不显示指示器）

## 📝 注意事项

1. **图片 URL 必须是 HTTPS**
   - 代码会自动将 `http://` 转换为 `https://`

2. **图片加载失败处理**
   - 如果图片加载失败，会显示空白区域
   - 建议后端提供有效的图片 URL

3. **性能优化**
   - 图片使用 `mode="aspectFill"` 模式，自动裁剪适配
   - 轮播组件设置 `autoplay="{{false}}"`，避免自动播放

4. **兼容性**
   - 支持微信小程序基础库 2.0.0 及以上
   - `backdrop-filter` 需要基础库 2.9.0 及以上

## 🔧 自定义配置

### 修改图片高度

在 `index.wxss` 中修改：

```css
.image-swiper-container {
  height: 500rpx;  /* 修改这里 */
}
```

### 修改最大图片数量

在 `index.js` 的 `extractImages()` 方法中修改：

```javascript
.slice(0, 9);  // 修改这里，例如改成 .slice(0, 6) 最多显示6张
```

### 修改指示器样式

在 `index.wxss` 中修改：

```css
.image-indicator {
  background: rgba(0, 0, 0, 0.6);  /* 背景色 */
  color: #fff;  /* 文字颜色 */
  font-size: 22rpx;  /* 字体大小 */
}
```
