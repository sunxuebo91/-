## 🎯 核心功能

### 自动加载文章
- 页面加载时自动调用 `loadArticles()`
- 默认获取6篇已发布的文章
- 无文章时自动隐藏模块

### 数据格式化
```javascript
{
  _id: "文章ID",
  title: "文章标题",
  author: "作者",
  coverImage: "封面图URL",
  viewCount: 100
}
```

### 点击交互
- 点击卡片有缩放动画（scale 0.98）
- 点击后显示"文章详情页开发中"提示
- 预留了详情页跳转逻辑

## 🎨 设计特点

1. **2列网格布局** - 充分利用屏幕空间
2. **圆角卡片** - 12rpx圆角，现代化设计
3. **封面图固定高度** - 200rpx，保持一致性
4. **标题自动截断** - 最多显示2行，超出显示省略号
5. **分隔线** - 与四个入口按钮用细线分隔
6. **响应式** - 自适应不同屏幕尺寸

## 📊 API调用

### 接口地址
```
GET https://crm.andejiazheng.com/api/articles?status=published&page=1&pageSize=6
```

### 请求参数
- `status`: published（只获取已发布文章）
- `page`: 1（第一页）
- `pageSize`: 6（每页6篇）

### 返回数据
```json
{
  "success": true,
  "data": {
    "list": [...],
    "total": 50,
    "page": 1,
    "pageSize": 6,
    "totalPages": 9
  }
}
```
