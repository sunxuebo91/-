## 🗄️ 阶段二：数据库设计（1天）

### 任务 2.1：创建 babyDiaries 集合

**字段清单**：
```javascript
{
  _id: String,              // 主键
  contractId: String,       // 合同ID（索引）
  customerId: String,       // 客户ID（索引）
  nurseId: String,          // 月嫂ID（索引）
  serviceDate: String,      // 服务日期（索引）
  dayNumber: Number,        // 服务第几天
  baby: Object,             // 宝宝记录
  mother: Object,           // 宝妈记录
  status: String,           // 状态（draft/published）
  createdAt: Date,          // 创建时间
  updatedAt: Date,          // 更新时间
  createdBy: String,        // 创建者openid
}
```

**开发步骤**：
- [ ] 在云开发控制台创建 babyDiaries 集合
- [ ] 设置集合权限（仅创建者和管理员可读写）
- [ ] 创建索引：
  - [ ] contractId（单字段索引）
  - [ ] nurseId + serviceDate（复合索引）
  - [ ] customerId + serviceDate（复合索引）
- [ ] 导入测试数据（3-5条）
