## 📋 关键代码分析

### 1. 登录页面 (pages/login/index.js)

**核心功能：**
- 输入用户名和密码
- 调用登录 API
- 保存 Token 和用户信息
- 跳转到首页

**关键代码：**
```javascript
async onLogin() {
  const { username, password } = this.data;
  
  // 调用登录 API
  const response = await login(username.trim(), password.trim());
  
  if (response && response.success && response.data) {
    const token = response.data.access_token;
    
    // 保存 Token（双键存储，兼容性）
    wx.setStorageSync('access_token', token);
    wx.setStorageSync('token', token);
    
    // 保存用户信息
    const userInfo = {
      id: response.data.user?.id,
      name: response.data.user?.name,
      role: response.data.user?.role,
      permissions: response.data.user?.permissions,
      ...response.data.user
    };
    wx.setStorageSync('userInfo', userInfo);
    
    // 跳转到首页
    wx.switchTab({ url: '/pages/home/index' });
  }
}
```

### 3. Request 工具 (utils/request.js)

**核心功能：**
- 封装 HTTP 请求
- 自动添加 JWT Token
- 处理 401 错误（Token 过期）

**关键代码：**
```javascript
const BASE_URL = 'https://crm.andejiazheng.com/api';

// 公开请求（无需 Token）
const publicRequest = (options) => {
  return new Promise((resolve, reject) => {
    wx.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data || {},
      header: {
        'Content-Type': 'application/json',
        'X-Client-Type': 'miniprogram',
        'X-Platform': 'wechat'
      },
      success: (response) => {
        if (response.statusCode === 200) {
          resolve(response.data);
        } else {
          reject(new Error(response.data?.message || '请求失败'));
        }
      },
      fail: reject
    });
  });
};

// 认证请求（需要 Token）
const authenticatedRequest = (options) => {
  const token = wx.getStorageSync('access_token');
  
  return new Promise((resolve, reject) => {
    wx.request({
      url: BASE_URL + options.url,
      method: options.method || 'GET',
      data: options.data || {},
      header: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${token}`,
        'X-Client-Type': 'miniprogram',
        'X-Platform': 'wechat'
      },
      success: (response) => {
        if (response.statusCode === 401) {
          // Token 过期，清除本地数据并跳转到登录页
          wx.removeStorageSync('access_token');
          wx.removeStorageSync('userInfo');
          wx.reLaunch({ url: '/pages/login/index' });
          reject(new Error('登录已过期'));
        } else if (response.statusCode === 200) {
          resolve(response.data);
        } else {
          reject(new Error(response.data?.message || '请求失败'));
        }
      },
      fail: reject
    });
  });
};
```

## 🔐 权限系统设计

### 用户角色

根据"安得家政"的设计，用户信息包含：

```javascript
{
  id: "user_123",
  name: "张三",
  username: "zhangsan",
  phone: "13800138000",
  role: "admin",  // admin, manager, advisor, staff
  permissions: ["customer:view", "customer:create", "customer:edit"],
  department: "销售部"
}
```

### 权限检查

在页面中检查权限：

```javascript
Page({
  onLoad() {
    const userInfo = wx.getStorageSync('userInfo');
    
    // 检查是否登录
    if (!userInfo) {
      wx.redirectTo({ url: '/pages/login/index' });
      return;
    }
    
    // 检查角色权限
    if (!['admin', 'manager'].includes(userInfo.role)) {
      wx.showToast({
        title: '无权限访问',
        icon: 'none'
      });
      wx.navigateBack();
      return;
    }
  }
});
```

## ⚠️ 注意事项

### 1. 域名配置

需要在小程序后台配置服务器域名：
- `https://crm.andejiazheng.com`

### 2. Token 过期处理

Token 过期时自动跳转到登录页：

```javascript
if (response.statusCode === 401) {
  wx.removeStorageSync('access_token');
  wx.removeStorageSync('userInfo');
  wx.reLaunch({ url: '/pages/login/index' });
}
```

### 3. 兼容性

为了兼容性，Token 使用双键存储：
- `access_token`（主键）
- `token`（备用键）
