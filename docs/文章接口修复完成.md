小程序调用的接口路径**不正确**：

```javascript

// 详情接口
GET /api/articles/:id
```

```javascript

GET /api/articles/miniprogram/list?page=1&pageSize=10

// 详情接口

```

## 📝 接口说明

### 获取文章列表

**接口**: `GET /api/articles/miniprogram/list`

**参数**:
- `page`: 页码（默认 1）
- `pageSize`: 每页数量（默认 10）
- `keyword`: 搜索关键词（可选）

**返回**:
```json
{
  "success": true,
  "data": {
    "list": [
      {
        "_id": "文章ID",
        "title": "文章标题",
        "author": "作者",
        "source": "来源",
        "summary": "摘要",
        "coverImage": "封面图URL",
        "imageUrls": ["图片1", "图片2"],
        "contentHtml": "<p>富文本内容</p>",
        "publishedAt": "2024-01-01T00:00:00.000Z",
        "viewCount": 100
      }
    ],
    "total": 4,
    "page": 1,
    "pageSize": 10,
    "totalPages": 1
  }
}
```

### 获取文章详情

**接口**: `GET /api/articles/miniprogram/:id`

**参数**:
- `id`: 文章ID（路径参数）

**返回**:
```json
{
  "success": true,
  "data": {
    "_id": "文章ID",
    "title": "文章标题",
    "author": "作者",
    "source": "来源",
    "summary": "摘要",
    "contentHtml": "<p>富文本内容</p>",
    "imageUrls": ["图片1", "图片2"],
    "coverImage": "封面图URL",
    "publishedAt": "2024-01-01T00:00:00.000Z",
    "viewCount": 100,
    "status": "published"
  }
}
```
